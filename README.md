## Setup

```bash
git clone git@github.com:MikRyam/todosApp.git

npm install
# or
yarn install
```

## Getting Started

```bash
npm run start
# or
yarn start
```

## Other available Scripts

```bash
# run ESLint
yarn lint

# run Prettier
yarn format

# run tests
yarn test
```

## Bundle application

```bash
# run build
yarn build
```
